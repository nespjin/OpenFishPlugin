│  │

│  │
│  ├─runtime
│  │
│  └─movie-runtime 小丑鱼影视实现

├─linux Linux平台实现(暂未开发)
├─macos MacOS平台实现(暂未开发)
├─native Native平台实现(暂未开发)
├─sdk 插件开发工具包以及文档(开发中...)[插件开发请查阅此内容]
├─tools 开发工具
└─windows Windows平台实现(暂未开发)
